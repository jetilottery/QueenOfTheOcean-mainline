A Dama dos Mares

Está pronto para um jogo de combinação de 3 símbolos? A Dama dos Mares concede seis rodadas para coletar as gemas para cada aposta.

Tente obter 3 gemas idênticas em uma rodada para ganhar o prêmio associado. Se obtiver uma gema em formato de estrela, você ganhará instantaneamente o prêmio associado.

Como apostar

Valor da aposta

Exibe o valor atual da aposta.

Pressione a seta para a esquerda (-) para diminuir o valor da aposta.

Pressione a seta para a direita (+) direita para aumentar o valor da aposta.

Pressione COMPRAR para deduzir o valor da aposta do saldo e jogar.

Revelar Tudo

Pressione REVELAR TUDO para que o jogo seja concluído automaticamente. O jogo continua até que todas as seleções sejam feitas.

Como jogar

O objetivo de A Dama dos Mares é encontrar 3 gemas idênticas em uma rodada. Encontre uma gema em formato de estrela em uma rodada e ganhe o prêmio correspondente instantaneamente.

No começo do jogo, A Dama dos Mares mostra 6 rodadas, cada uma com 5 mapas. Selecione um mapa para revelar uma gema, e use REVELAR TUDO para que o jogo seja concluído automaticamente.

Se o jogador encontrar:

  • 3 gemas idênticas em uma rodada, ganhará o prêmio da rodada.
  • Uma gema em forma de estrela, ganhará o prêmio da rodada.

Para concluir o jogo, selecione todos os mapas camisas ou use REVELAR TUDO para exibir uma mensagem indicando o resultado, encerrando o jogo.

Regras

Uso inadequado ou mau funcionamento invalida todos os pagamentos e jogos.

Os prêmios são exibidos em moeda corrente.
